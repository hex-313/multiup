Tutorial to watch premium streaming video

To watch premium steaming video, in debrid page or mirror page, you need to use your prefered media player, such as VLC and install a special web-browser plugin.

  • Download and install "External Application Button" plugin for your web-browser :
    • Chrome plugin
    • FireFox plugin
    • Opera plugin
  • Download native client for your operating system : "windows.zip" or "mac.zip" or "linux.zip" : Native client
  • Extract zip file
  • Open "install.bat" or "install.sh" to install native client
  • Go to addons section in your web-browser
  • Click on preferences for "External Application Button"
  • Edit addon parameters :
    • Display name : write "Watch video in streaming"
    • Executable name : write media player's path (ex: C:\Program Files (x86)\VideoLAN\VLC\vlc.exe)
    • Arguments : write "[HREF]"
    • Placement / Context Menu : tick "Link Context"
    • Icon : choose an icon in your computer (you can also download : MultiUp icon)
    • Click on : "Add Application"
  • You can now watch streaming video in your media player from debrid page or mirror page
  • Just right click on "Download premium" and select "Watch video in streaming"

Tutorial to upload

First method (easy)
  • Go to "Upload" page => http://multiup.org/en/upload/from-computer
  • Select "Upload JQuery"
  • Unselect files hosting you doesn't want or leave it as it is
  • Click on the green "Add Files..." button
  • Select your files into your computer
  • Click "Start Upload" button
  • You can see the upload speed and time remaining
  • When upload is finished, you will be redirected to the linking page
  • Just copy link and share it !
Second method (easy)
  • Go to "Upload" page => http://multiup.org/en/upload/from-computer
  • Select "Upload with Softwares"
  • Click on "Download MultiUp Manager"
  • Unzip the file
  • Open "MultiUp.exe"
  • Unselect files hosting you doesn't want or leave it as it is
  • Click on "Select your files"
  • Click on "Start Upload !"
  • You can see the upload speed and time remaining
  • When your file is uploaded, click on "Show links"
  • You can add your multiup account into "Options" tab (Login and Password)
  • Press "Connexion" button to be connected to your account
  • Then Start Upload
Third solution (average)
  • Go to "Upload" page => http://multiup.org/en/upload/from-computer
  • Select "Upload Remote"
  • Unselect files hosting you doesn't want or leave it as it is
  • Put your remote link into the white case
  • Click on "Start Upload" button
  • You can see the upload speed and time remaining
  • When upload is finished, you will be redirected to the linking page
  • Just copy link and share it !
